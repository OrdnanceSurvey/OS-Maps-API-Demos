OS OpenSpace API

The OS OpenSpace© Application Programming Interface (API) is build using a code library called OpenLayers that enables you to embed and use our maps in business and non-business website or mobile applications.

An API (Application programming interface) is a defined set of protocols and responses, allowing web communities to create an open architecture for sharing content and data between communities and applications

Take a look at our getting started guide, sample code and documentation to see how to start building your first application and to get the most out of OS OpenSpace© or take a look at the gallery to see other aplications.
